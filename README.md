## 🧰 Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/nodoubtz/Nodoubtz-AI.git
   cd Nodoubtz-AI
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```
